int alphabeta ( Map &  map,
int  turn,
int  depth,
bool  isPlayingPhase,
int  alpha,
int  beta 
)

Execute minimax with alphabeta pruning to a specified depth. If the timer for the move runs out, 0 is returned immediately. For alphabeta pruning every node has the information which is the minimal(in case of minimizer nodes) or the maximal(in case of maximizer nodes) value of its parent node. While building its subtree, it constantly checks whether its value is higher than the reference value(if parent=minimizer) or lower otherwise. If this is the case, the node returns immediatly and the subtree is pruned away.

Parameters
map- The initial game state
turn- The player who has to make a move
depth- Specifies to what depth the game tree should be built
isPlayingPhase- Specifies in which phase of the game currently is active
alpha- The currently highest value of a parent maximizer
beta- The currently lowest value of a parant minimizer
Returns
The minimal value of all child nodes in case of minimizer, the maximum value otherwise

int minimax ( Map &  map,
int  turn,
int  depth,
bool  isPlayingPhase 
)

Executing minimax with paranoid strategy. The board will be analyzed for the given player at the leafs of the game tree. At each node the value shall be maximized for the given player turn, minimized for others. A player's turn is represented by the branches leaving that specific node.

Parameters
map- The current board state
turn- The player who has to make a move
depth- How deep(amount of successive moves) the algorithm should look from a specific node
isPlayingPhase- Set if we are in the playing phase or bombing phase
Returns
The evaluation value of the best move for that node
