Move* getNextMove ( Map &  map,
int  searchDepth,
int  searchTime,
int  player,
int  algorithm,
bool  isPlayingPhase 
)

The function will look for the best move it can find with the given search depth or time limit constraints. Note that if both limits are set, only the depth limit will apply and the time limit will be set to its default of 90 seconds.

Parameters
map- The initial state of the board
searchDepth- The depth up to which the search tree should be build. Set to 0 to use a time limit.
searchTime- The time limit set for the move search. Does not apply if searchDepth has been set.
player- The player who can make a move.
algorithm- The identifier for the algorithm that should be used
isPlayeringPhase- Used to distinguish between playing and bombing phase
Returns
A pointer to the move identified as the best move.
